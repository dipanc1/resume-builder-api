
# Resume Builder API

NestJS APIs for [AICVPro](https://aicvpro.com)


## Run Locally

Clone the project

```bash
  git clone https://github.com/dipanc1/resume-builder-api.git
```

Go to the project directory

```bash
  cd resume-builder-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
